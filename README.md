# ArgonAssault

A simple unity game which uses Terrain.

### Play Controls

<b>Movement :</b> &nbsp; WASD / Arrow Keys <br/>
<b>Fire :</b> &nbsp; Left Mouse Click / Left Ctrl <br/>
<b>Exit Game :</b> &nbsp; Esc <br/> <br/>
